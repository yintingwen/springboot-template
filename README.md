# 包路径命名规范
* 单模块：com.域名.项目
* 多模块：
    * com.域名.项目.模块
    * com.项目.模块

# 单模块POM命名
* \<groupId>com.域名\</groupId>
* \<artifactId>项目\</artifactId>

# 多模块POM命名1
* \<groupId>com.域名\</groupId>
* \<artifactId>项目-模块\</artifactId>

# 多模块POM命名2
* \<groupId>com.域名.项目\</groupId>
* \<artifactId>项目-模块\</artifactId>
